# Bounce-Color-Box

このゲームは元々Scratchで製作したものを移植かつ様々な要素を変更&追加し､快適にプレイできるように調整しました｡


### ■Scratch版からの変更点
・高解像度化(1080p)

・高フレームレートの対応(90,120fps)

・画面比率を固定(16:9)

・影描画モードなどの一部設定項目の廃止

・多少の挙動の変更

・収集要素の内容や一部スプライトの変更

・UIの改善&フォントの変更

・データロードするとセーブした地点から始めることができるようになりました｡

・メスモードの追加

・タイマー機能の追加

### ■開発背景
この作品は元々実験的に製作したものであり､今回もUnityでどんなことができるのかという実験的なところがあります｡ただ､当初はUnityに標準で搭載されている物理エンジンを使用する予定だったものの､技術不足ということや原作のような挙動を再現することが困難になったこともあり､自ら物理エンジンをコーディングすることで解決しました｡この作品を実際に製作してみて今後のゲーム製作のノウハウに活かせればと思います｡
