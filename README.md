# Bounce-Color-Box


このゲームは元々Scratchで製作したものを移植かつ様々な要素を変更&追加し､快適にプレイできるように調整しました｡

## Scratch版からの変更点
・高解像度化(1080p)
・高フレームレート化(60fps以上対応)
・影描画モードなどの一部設定項目の廃止
・多少の挙動の変更
・収集要素の内容や一部スプライトの変更
・UIの改善&フォントの変更
・データロードするとセーブした地点から始めることができるようになりました｡
・メスモードの追加
・タイマー機能の追加

##開発背景
この作品は元々実験的に製作したものであり､今回もUnityでどんなことができるのかという実験的なところがあります｡この作品を実際に製作してみて今後のゲーム製作のノウハウに活かせればと思います｡
